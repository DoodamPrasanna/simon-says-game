# 🎮 Simon Says Game

A fun and interactive **Simon Says Memory Game** built using HTML, CSS, and JavaScript.  
The game challenges players to remember and repeat an increasing sequence of colors. As the levels progress, the difficulty increases, testing memory and concentration skills.
